# Challenge 5: Content Performance Prediction

## Business Problem
A digital media company wants to improve engagement by predicting how well content will perform before it is published.

This would allow creators to optimize content strategy.

---

## Data
Here is an example of given content data, including:

- `post_time`: Time content was posted
- `content_type`: Type of content (image, video, etc.)
- `caption_length`: Length of caption
- `hashtags_count`: Number of hashtags used
- `follower_count`: Number of followers of the creator
- `engagement_score`: Overall engagement metric

Use this as a frame of reference when looking through the kaggle dataset.

Name of Dataset: Instagram Analytics Dataset  
KAGGLE CLI DOWNLOAD: kaggle datasets download -d kundanbedmutha/instagram-analytics-dataset  
Kaggle Link: https://www.kaggle.com/datasets/kundanbedmutha/instagram-analytics-dataset

---

## Tasks
- Analyze which factors influence content performance
- Identify patterns across content types and posting times
- Build a model to predict engagement score
- Evaluate model performance
- Identify the most important factors driving engagement
