# Sprinternship Advanced Data Science & AI Workshop

## Goal
You will work in teams on a mock industry challenge using the CRISP-DM lifecycle:
1. Business Understanding
2. Data Understanding
3. Data Preparation
4. Modeling
5. Evaluation
6. Recommendations

## Repository Structure
Each challenge will have its own designated folder with it's problem statement and data. cd into your challenge project to access your data and read over the business problem and tasks
- `cd` into your designated folder
- For example:
- `cd 1_ClinicalPriority`
- you can `ls` to check your directories
- `cd ..` if you want to redo

## Setup
1. Clone this repo
2. Create a virtual environment
3. Install dependencies
4. Open your assigned challenge

## Deliverable
Each team should present:
- Problem framing
- Key data insights
- Modeling approach
- Evaluation results
- Business recommendations
